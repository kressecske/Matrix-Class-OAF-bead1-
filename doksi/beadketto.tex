%%%%%%%%%%%%%%%%%%%% FELADAT LEÍRÁSA %%%%%%%%%%%%%%%%%%%%
{\textbf {Feladatleírás}}\\[2pt]
Valósítsa meg az egész számokat tartalmazó felsõháromszög mátrixtípust (a
mátrixok a fõátlójuk alatt csak nullát tartalmaznak)! Ilyenkor elegendõ csak a fõátló
és afeletti elemeket reprezentálni egy sorozatban, amelyet egy dinamikus
helyfoglalású tömbben helyezzünk el. Implementálja önálló metódusként a mátrix iedik
sorának j-edik elemét visszaadó mûveletet, valamint hatékony összeadás és
szorzás mûveleteket, továbbá a mátrix (négyzetes alakú) kiírását, és végül a másoló
konstruktort és az értékadás operátort!

%%%%%%%%%%%%%%%%%%%% SPECIFIKÁCIÓ %%%%%%%%%%%%%%%%%%%%
{\textbf {Típusérték-halmaz}}\\[2pt]

 Matrix(m) = \{a $\in \mathbb{Z}^{m*m} $ | m $\in \mathbb{N} \wedge \forall i,j \in [1..m]: i>j \rightarrow a[i][j]=0$ \}\\

{\textbf {Típus mûveletek}}\\[2pt]


\textit{1.Lekérdezés}

\hspace{18pt}$\!A=\alatt{Matrix(\_meret)}{m}\times \alatt{\mathbb{Z}}{i}\times \alatt{\mathbb{Z}}{j}\times \alatt{\mathbb{Z}}{e}$\\[4pt]


\hspace{18pt}$\!Q = (m=m' \wedge i=i' \wedge j=j' \wedge i,j \in [0..\_meret])$\\[4pt]

\hspace{18pt}$\!R = ( Q \wedge e=a[i][j] )$\\[4pt]

\textit{2.Felülírás}

\hspace{18pt}$\!A=\alatt{Matrix(\_meret)}{m}\times \alatt{\mathbb{Z}}{i}\times \alatt{\mathbb{Z}}{j}\times \alatt{\mathbb{Z}}{szam}$\\[4pt]


\hspace{18pt}$\!Q = (m=m' \wedge i=i' \wedge j=j' \wedge i,j \in [1..\_meret] \wedge szam=szam' \wedge i\leq j)$\\[4pt]

\hspace{18pt}$\!R = ( i=i' \wedge j=j' \wedge a[i][j]=szam \wedge \forall k,l \in [1..\_meret]: (k\neq i \vee l \neq j) \rightarrow a[k,l]=a'[k,l] )$\\[4pt]

\textit{3.Összeadás}

\hspace{18pt}$\!A=\alatt{Matrix(\_meret)}{a}\times\alatt{Matrix(\_meret)}{b}\times\alatt{Matrix(\_meret)}{c}$\\[4pt]


\hspace{18pt}$\!Q = (a=a' \wedge b=b')$\\[4pt]

\hspace{18pt}$\!R = ( Q \wedge \forall i,j \in [1.._meret] : c[i][j]= a[i][j] + b[i][j] )$\\[4pt]

\textit{4.Szorzás}

\hspace{18pt}$\!A=\alatt{Matrix(\_meret)}{a}\times\alatt{Matrix(\_meret)}{b}\times\alatt{Matrix(\_meret)}{c}$\\[4pt]


\hspace{18pt}$\!Q = (a=a' \wedge b=b')$\\[4pt]

\hspace{18pt}$\!R = ( Q \wedge \forall i,j \in [1..\_meret] : c[i][j]= a[i][j] * b[i][j] )$\\[4pt]

{\textbf {Reprezentáció}}\\[2pt]
Az n*n-es felsõháromszög mátrixot (csak a fõátló és felett tartalmaz nem nulla elemeket), az egész mátrixot ábrázolva valósítjuk meg a programba, ügyelve arra, hogy a fõátló alatti elemek mind nulla.\\
\newpage
{\textbf {Implementáció}}\\[2pt]
\textit{1.Lekérdezés}\\
Az m mátrix i-edik sorának j-edik elemét visszaadó e:=m[i][j] értékadás programmal implementálható feltéve, hogy i $\leq j$ és i,j $\in$ [1..n],ahol n a mátrix mérete.
\begin{stuki}[5cm]
				\begin{IF}{1}
					{\stm{ i \leq j}}
					\stm{ e:= m[i][j]}
					\ELSE
					\stm{-}
				\end{IF}
\end{stuki}
\textit{2.Felülírás}
Az m mátrix i-edik sorának j-edik elemét felülíró m[i][j]:=szam értékadás programmal implementálható feltéve, hogy i $\leq j$ és i,j $\in$ [1..n],ahol n a mátrix mérete.
\begin{stuki}[6cm]
	\begin{IF}{1}
		{\stm{ i \leq j}}
		\stm{ m[i][j]= szam}
		\ELSE
		\stm{-}
	\end{IF}
\end{stuki}
\textit{3.Összeadás}\\
$\forall i,j \in [1..\_meret] : c[i][j]= a[i][j] + b[i][j] $\\[4pt]
\textit{4.Szorzás}\\
$\forall i,j \in [1..\_meret] : c[i][j]= a[i][j] * b[i][j] $\\[16pt]

%%%%%%%%%%%%%%%%%%%% TESZTELÉSI TERV %%%%%%%%%%%%%%%%%%%%
{\textbf {Tesztelési terv:}}\\[2pt]
\begin{enumerate}[topsep=2pt,noitemsep,label=\arabic*),leftmargin=*]
\item Különféle méretû mátrixok létrehozása, feltöltése és kiírása
	\begin{enumerate}[topsep=2pt,noitemsep,label=\alph*),leftmargin=*]
		\item 0,1,2,5 dimenziójú mátrix
		\item Az a+b illetve a*b kifejezés kiíratása		
	\end{enumerate}
\item Mátrix adott pozíciójú értékének lekérdezése és megváltoztatása. 
	\begin{enumerate}[topsep=2pt,noitemsep,label=\alph*),leftmargin=*]
		\item Hibás pozíció megadása.
		\item Felsõ-háromszög mátrixba esõ elem lekérdezése
		\item Felsõ-háromszög mátrixba esõ elem megváltoztatása.
		\item Felsõ-háromszög mátrixon kívül esõ elem lekérdezése
		\item Felsõ-háromszög mátrixon kívül esõ elem megváltoztatása	
	\end{enumerate}
\item A c:=a+b mátrixösszeadás kipróbálása
	\begin{enumerate}[topsep=2pt,noitemsep,label=\alph*),leftmargin=*]
		\item Eltérõ méretõ mátrixokkal (az a és b mérete különbözik, a c és a mérete különbözik)
		\item Kommutatívitás ellenõrzése (a + b == b + a)
		\item Asszociatívitás ellenõrzése (a + b + c == (a + b) + c == a + (b + c))
		\item Null elem vizsgálata (a + 0 == a, ahol 0 a null mátrix)
	\end{enumerate}

\item A c:=a*b mátrixszorzás kipróbálása
	\begin{enumerate}[topsep=2pt,noitemsep,label=\alph*),leftmargin=*]
		\item Eltérõ méretõ mátrixokkal. (az a és b mérete különbözik, a c és a mérete különbözik)
		\item Kommutatívitás ellenõrzése (a * b == b * a)
		\item Asszociatívitás ellenõrzése (a * b * c == (a * b) * c == a * (b * c))
		\item Null elem vizsgálata (a * 0 == 0, ahol 0 a null mátrix)
	\end{enumerate}




\end{enumerate}
